## Config / Instalação

Criando um ambiente virtual

```bash
  poetry install
  poetry shell
```

Rodando Todos os Tests

```bash
  pytest -vv
```

Rodando Apenas Test Unitarios

```bash
  pytest -m 'unit'
```

Rodando Apenas Test Funcionais

```bash
  pytest -m 'functional'
```
